# Strawberry GraphQL

A Python GraphQL library using code-first schema creation with Python type hints and dataclasses. Strawberry supports ASGI/WSGI frameworks, subscriptions, federation, dataloaders, and schema directives, delivering more than 6 million downloads per month.

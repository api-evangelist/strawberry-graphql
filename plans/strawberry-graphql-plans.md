# Strawberry GraphQL Plans

Strawberry GraphQL is an open-source Python library released under the MIT license. There are no commercial tiers or paid plans for the library itself — it is free to use, modify, and distribute.

## Open Source (Free)

- Full access to the Strawberry GraphQL library via PyPI (`pip install strawberry-graphql`)
- All features included: queries, mutations, subscriptions, federation, dataloaders, schema directives, extensions
- Support for Django, Flask, FastAPI, Litestar, Sanic, Channels, and other ASGI/WSGI frameworks
- Community support via Discord (https://strawberry.rocks/discord)
- Issue tracking and contributions via GitHub (https://github.com/strawberry-graphql/strawberry)

## Sponsorship / Open Collective

Strawberry GraphQL accepts financial contributions through Open Collective to sustain ongoing development:

- **Individual Backers** — one-time or recurring donations at any amount
- **Organization Sponsors** — recurring monthly contributions; sponsor logos may appear in project documentation and the website

Sponsorship details: https://opencollective.com/strawberry-graphql

## Notes

- No SLA, enterprise support tier, or hosted service is offered directly by the Strawberry GraphQL project.
- Hosted GraphQL services or commercial support may be available through third-party vendors integrating Strawberry.
